> HyperX는 Unity 2021 LTS 이상의 .NET 호환 환경에서 원할하게 동작합니다.

<Warning>Unity를 위한 .NET SDK는 현재 Early Access 단계이며, 라이브러리 파일을 다운로드 받기 위해 별도의 Github 권한을 받아야합니다. 다운로드 페이지 접근 권한이 없는 경우 권한 부여를 요청해주세요.</Warning>

## Unity 프로젝트에 HyperX 설치하기

### Step 1: 라이브러리 파일 다운로드

아래 링크에 접속하여 가장 최신 버전 릴리즈의 `HyperX.netstandard2.1.dll` 파일을 다운로드 받으세요.
- https://github.com/team-hyperx/hyperx-dotnet-sdk/releases

### Step 2: dll 파일을 Unity 프로젝트에 불러오기

다운로드 받은 dll 파일을 유니티 프로젝트의 Plugins 폴더에 넣어주세요.
![](/images/quickstart-unity/1.png)

### Step 3: 프로젝트 초기화 코드 실행

스플래시나 인트로 화면 등에서 아래 코드가 실행 될 수 있도록 해주세요.
```cs
HyperX.Core.Init("프로젝트코드");
```

#### 프로젝트 코드란 무엇인가요?

콘솔에서 프로젝트를 선택했을 때 URL에 명시되어있는 프로젝트 식별자가 프로젝트 코드입니다.
- 아래와 같은 사진에서 `HyperX's Project` 프로젝트의 프로젝트 코드는 `0199c85f-8f24-762c-a65f-de45bac2a3c5`입니다.

![](/images/quickstart-unity/2.png)

### Step 4: 빌드 설정에서 .NET 호환 버전 설정하기

유니티 프로젝트 빌드 설정에서 `Api Compatibility Level`을 `.NET Standard 2.1` 이상으로 설정해주세요.

![](/images/quickstart-unity/3.png)
